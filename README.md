# project
sprint 7

Por medio de un conjunto de datos de anuncios de venta de coches se procedera a crear una pagina web donde se pueda visualizar un histograma y un grafico de dispersion.

Por medio del histograma podremos tener una idea de cual es la frecuencia de kilometros recorridos por los autos.

Por medio del grafico de dispersion podremos tener una idea de el recorrido de los kilometros del carro dependiendo del precio de este mismo
